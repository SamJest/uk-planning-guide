When This Project Hub Is The Right Starting Point And When To Go More Local

Best when

  • You already know the build type and want the shortest route into the right local pages.
  • You need to compare areas before committing to one council path.
  • You want to isolate which planning rule is most likely to change the answer.

Go more local when

  • One council area, one conservation issue or one exact site constraint is already the live blocker.
  • The project is close to a limit and a generic hub is no longer specific enough.
  • You need a route decision, not a navigation layer.

What usually settles it faster

  • Open a local project page first if the town is already known.
  • Use the area hub if you are still comparing councils.
  • Use the topic hub if one planning rule is clearly doing most of the work.
